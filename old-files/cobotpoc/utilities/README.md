# Utilities

Small helper scripts for calibration and hardware sanity checks:

- `calibrate.py`: Captures matching robot-frame (mm) and camera-frame (px) points to compute homography.
- `eyedropper.py`: Opens a live camera feed and prints clicked pixel HSV values to tune vision color thresholds.
- `moverobot.py`: Simple robot motion test that toggles between two poses to verify MyCobot connection and basic movement.
